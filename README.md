# 💻 Formação Frontend: HTML5 & CSS3

Este repositório reúne todos os exercícios práticos, desafios de design responsivo e projetos reais que desenvolvi ao longo dos **5 módulos (200 horas)** do curso síncrono de HTML5 e CSS3 do Curso em Vídeo, ministrado pelo prof. Gustavo Guanabara.

Aqui demonstro minha sólida base em estruturação semântica, estilização avançada e adaptação de telas para dispositivos móveis (Mobile First).

---

## 🛠️ O que foi praticado e dominado aqui?

* **HTML5 Semântico:** Criação de estruturas otimizadas para SEO (motores de busca do Google).
* **CSS3 Moderno:** Domínio de layouts com Flexbox e CSS Grid para páginas organizadas.
* **Design Responsivo:** Uso de Media Queries para garantir que os sites funcionem perfeitamente em celulares, tablets e computadores.
* **Tipografia e Estética:** Aplicação prática de identidades visuais de alta qualidade.
